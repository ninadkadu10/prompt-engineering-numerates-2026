---

## What You're Building

A single, well-crafted prompt that turns ChatGPT into a personalised study assistant tailored to YOU.

**It must include all 5 components:**

| Component | Question to ask yourself |
|---|---|
| **Role** | What kind of expert should the AI be? |
| **Context** | What's your branch, year, subject, and current level? |
| **Task** | What exactly do you want it to help you with? |
| **Format** | How should it respond? (bullet points, quiz, table, explanation?) |
| **Constraints** | Any limits? (length, difficulty level, language style, examples?) |

## Step-by-Step Instructions

### Step 1 — Pick your subject (1 minute)
Choose ONE subject from your current semester that you're actually struggling with or want to revise before exams.

Write it here: _______________________________________________

### Step 2 — Fill in the prompt skeleton (2 minutes)

Use this template as a starting point — **replace every `[ ]` with your own words:**

```
Act as a [type of teacher/expert] who specialises in teaching [your subject]
to [your branch] students in their [year] year.

I currently understand [what you know] but I struggle with [what's hard for you].

Help me [specific task — revise / understand / practice / summarise] the topic of [specific topic].

Format your response as [format: bullet points / a quiz / a step-by-step breakdown / a cheat sheet / a story analogy].

Keep the explanation [difficulty: simple / intermediate] and under [length: 200 / 300 / 500] words.
Include [1-2 / 3-4] examples that relate to [something familiar to you — cricket, coding, real life, etc.].
```

### Step 3 — Run it in ChatGPT (1 minute)
Paste your prompt into ChatGPT and run it.  
Read the output. Ask yourself: *"Would I actually use this to study tonight?"*

### Step 4 — Improve it once (1–2 minutes)
Make **one targeted improvement** to your prompt based on the output:
- Was the output too long? → Add a word limit constraint
- Was it too basic? → Raise the difficulty level in context
- Did it miss examples? → Add "include 2 real-world examples"
- Was the format wrong? → Change it to quiz / table / flashcards

## Sample Prompts by Branch

Pick the one closest to your branch for inspiration — but write your OWN version.

**CS / AI-ML:**
```
Act as a senior software engineer who tutors first-year CS students.
I understand Python basics (loops, variables, functions) but I struggle with
object-oriented programming — specifically what "self" does and why we use classes.

Explain OOP to me using 3 real-world analogies. Then show me one simple class
example in Python with comments on every line.

Keep the total response under 300 words. End with one question I should be able
to answer if I've understood this topic.
```

**AIDS / BCA:**
```
Act as a data analyst who teaches statistics to beginners.
I'm a 1st-year AIDS student who knows basic math but has never done
statistics formally.

Explain the difference between mean, median, and mode using an example
from cricket scores or Bollywood box office numbers.
Then give me a 5-question multiple choice quiz at the end.
Keep each explanation under 50 words.
```

**BSc Mathematics / Statistics:**
```
Act as a university maths professor who makes abstract concepts visual and intuitive.
I'm a BSc Math student who understands basic calculus but struggles with
understanding WHY integration represents area under a curve — not just how to compute it.

Give me an intuitive explanation (no formal proofs) using a real-world scenario.
Then show me the same concept in 3 steps: intuition → formula → example.
Keep the total response under 400 words.
```
